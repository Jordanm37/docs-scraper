All desktop apps now support a native drawing feature, similar to drawing mode in the mobile apps.

![How to active the drawing mode](/images/drawing-mode-button.webp)

The drawing feature can be used with your trackpad, an external pen tablet, or with a touchscreen laptop. You can use drawing mode to digitize handwritten equations, text, tables, and chemical diagrams into LaTeX, MS Word, and other formats for creating technical documents.
