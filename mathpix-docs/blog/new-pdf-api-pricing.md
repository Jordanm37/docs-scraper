We’re excited to announce a significant discount on our PDF API pricing! Thanks to recent improvements in efficiency, we’ve reduced our costs while maintaining the same high-quality performance. With this pricing update, we are passing those cost savings to our customers.

**Old Pricing**:

* $0.025/page (0-40K pages)
* $0.01/page (40K+ pages)

**New Pricing** (Effective December 1, 2024):

* $0.005/page (0-1M pages)
* $0.0035/page (1M+ pages)
