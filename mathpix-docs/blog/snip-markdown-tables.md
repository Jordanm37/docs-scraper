The desktop Snips apps and [Snip web app](https://snip.mathpix.com) now support converting images of tables into Markdown. You can take Snips of tables and use the result in any editor that supports Markdown, like [Typora](https://typora.io/), [HackMD](https://hackmd.io/), and [MacDown](https://macdown.uranusjr.com/).
