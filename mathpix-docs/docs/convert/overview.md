# What is Convert API?

[Convert API](/convert) is a web API for extracting text from images, digitizing PDFs, and supporting digital ink in your application. It works especially well for content containing scientific notation like math, chemistry, and physics. You can find example images with API responses [here](/docs/convert/examples).

![Convert API](/images/ocr-hero-desktop.webp)

# API reference

The Convert API developer docs are currently available at [docs.mathpix.com](https://docs.mathpix.com).
