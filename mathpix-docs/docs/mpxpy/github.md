# Mathpix Python SDK on GitHub

The source code for the Mathpix Python SDK (mpxpy) is publicly available on GitHub.

Repository URL: <https://github.com/Mathpix/mpxpy>

## What You’ll Find

* Full SDK source code.
* Open issues and discussions related to SDK development.
* Installation instructions for developers.
* Contribution guidelines if you wish to suggest improvements or submit pull requests.
* Changelog and version history (tracked via GitHub releases).

* Currently, there is no separate changelog published inside the Mathpix documentation.
* All updates and versioning are maintained directly through GitHub.
