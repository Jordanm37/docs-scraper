# Snip Chrome Extension

Our [Chrome Extension](https://mathpix.com/chrome-extension) makes it possible to convert PDFs directly from your browser. Your PDFs sync instantly across all of your devices so that you can add documents using the Chrome extension on the web and continue reading and editing on mobile or tablet.

# How it works

1. Install the extension [here](https://chrome.google.com/webstore/detail/mathpix-snip/acjfbklbdkhaamiiehfmmepaiddbnido).
2. Open a PDF in your browser and find the extension in the right upper corner.
3. Click “Add to my collection”.

Now a PDF is added to your searchable Snip repository and can be edited, read, or exported to Markdown, DOCX, HTML, and more.
