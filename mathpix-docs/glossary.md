# Feature Glossary

### [Chrome Extension](/chrome-extension)

The [Mathpix Snip Chrome extension](https://chrome.google.com/webstore/detail/snip/acjfbklbdkhaamiiehfmmepaiddbnido) makes it possible to upload a PDF in your browser to your PDF collection in Snip instantly.

### [CLI tool](/mpx-cli)

A command-line interface tool that allows you to run commands locally on your computer. Mathpix offers a command line tool called [MPX CLI](https://github.com/Mathpix/mpx-cli) for document format conversion of the files on your local machine.

### [Command-line interface](/mpx-cli)

A command-line interface (CLI) processes commands to a computer program in the form of lines of text. Mathpix offers a command line tool called [MPX CLI](https://github.com/Mathpix/mpx-cli) for document format conversion of the files on your local machine.

### [Convert PDF to DOCX](/pdf-to-word)

Converting PDF files to DOCX format. Using Snip, Convert API, or MPX CLI you can convert PDF files to DOCX format, which is editable in MS Word.

### [Convert PDF to HTML](/pdf-to-html)

Converting PDF files to HTML format. Using Snip, Convert API, or MPX CLI you can convert PDF files to HTML format, which you can render on the web.

### [Convert PDF to LaTeX](/pdf-to-latex)

Converting PDF files to LaTeX format. Using Snip, Convert API, or MPX CLI you can export the LaTeX as a .zip or open in Overleaf.

### [Convert PDF to Markdown](/pdf-to-markdown)

Converting PDF files to Markdown format. Using Snip, Convert API, or MPX CLI you can convert PDF files to [Mathpix Markdown](/mathpix-markdown). Perfect compatibility with Snip web app and VS Code.

### [Convert PDF to Word](/pdf-to-word)

Converting PDF files to Microsoft Word format. Using Snip, Convert API, or MPX CLI you can convert PDF files to a format editable in MS Word.

### [Document conversion](/pdf-conversion)

Converting documents between different file types, like PDF, DOCX, LaTeX, HTML, and Markdown.

### [Equation to LaTeX](/equation-to-latex)

Converting printed and handwritten math formula to LaTeX.

### [Equation to MS Word](/equation-to-word)

Converting printed and handwritten equations to MathML, which can be pasted directly into MS Word document.

### [Image conversion](/image-conversion)

Converting images of math, text, tables, and chemical diagrams to formats like LaTeX, MathML, DOCX, Markdown, and SMILES, which can be pasted directly to your favorite document editing tools.

### [Image to AsciiMath](/image-to-asciimath)

Converting images of math into AsciiMath format, which can be pasted directly into your workspace.

### [Image to ChemDraw](/image-to-chemdraw)

Converting images of chemical diagrams into SMILES format, which can be pasted directly into ChemDraw.

### [Image to CSV](/table-to-csv)

Converting images of tables into digital format, which can be exported as a CSV file and opened in any spreadsheet tool.

### [Image to Excel](/ocr-table)

Converting images of tables into TSV format, which can be pasted into any spreadsheet tool like MS Excel.

### [Image to Google Sheets](/ocr-table)

Converting images of tables into TSV format, which can be pasted into any spreadsheet tool like MS Excel.

### [Image to LaTeX](/image-to-latex)

Converting images (of math, text, or tables) into LaTeX code, which can be pasted into any LaTeX editor, like [Overleaf](/docs/snip/snip-with-overleaf).

### [Image to MathML](/)

Converting images (of math, text, or tables) into MathML. Our tools use computer vision to convert images into MathML code.

### [Image to SMILES](/image-to-smiles)

Converting printed and handwritten chemical diagrams into canonical SMILES code.

### [Image to TSV](/pdf-to-tsv)

Converting images of tables into TSV format, which can be pasted into any spreadsheet tool.

### [Image to Word](/image-to-word)

Converting images to text or math that can be pasted into an MS Word document.

### [LaTeX](/)

LaTeX is a software system for document preparation and is an alternative to WYSIWYG editors like MS Word. Many editors use "math mode" LaTeX for rendering math in documents. Our tools can convert images and PDFs into LaTeX.

### [LaTeX Figure Syntax](/docs/mathpix-markdown/figures)

Using figure environment to create figures, which can be aligned, captioned and labelled.

### [LaTeX List Syntax](/docs/mathpix-markdown/lists)

Using itemize LaTeX environment to create ordered, unordered, and nested lists.

### [LaTeX Table Syntax](/docs/mathpix-markdown/tables)

Using tabular environment to typeset tables with optional horizontal and vertical lines. Tables can be captioned, labelled and referenced.

### [Mathpix Markdown](/mathpix-markdown)

Markdown is a lightweight markup language for creating formatted text using a plain-text editor, created by John Gruber and Aaron Swartz. Mathpix provides its own superset of Markdown that has extended support for STEM features, called Mathpix Markdown, or MMD for short.

Mathpix Markdown is a superset of Markdown with extended support for LaTeX and other STEM-specific syntax like SMILES for chemical diagram rendering. An open source implementation called mathpix-markdown-it is [available on Github](https://github.com/Mathpix/mathpix-markdown-it).

### [v3/batch](https://docs.mathpix.com/#process-a-batch)

Endpoint to process a multiple images in a single POST request.

### [v3/latex](https://docs.mathpix.com/#process-an-equation-image)

Our legacy endpoint for converting images to LaTeX, that was developed before we added support for regular text. It should only be used for images that only have individual equations and no text that needs to be processed.

### [v3/ocr-results](https://docs.mathpix.com/#query-image-results)

Endpoint to search results from POST requests to v3/text, v3/strokes, and v3/latex with a GET request on `/v3/ocr-results?search-parameters`.

### [v3/pdf](https://docs.mathpix.com/#process-a-pdf)

Endpoint to OCR PDFs and convert them to Markdown, LaTeX, and DOCX.

### [v3/pdf-results](https://docs.mathpix.com/#query-pdf-results)

Endpoint to search results from POST requests to v3/pdf with a GET request on `/v3/pdf-results?search-parameters`.

### [v3/strokes](https://docs.mathpix.com/#process-stroke-data)

Endpoint to add stroke recognition (i.e. digital ink) to your app. Works for all handwritten math. Also supports regular text handwriting for Latin alphabets and Hindi.

### [v3/text](https://docs.mathpix.com/#process-an-image)

Endpoint to OCR individual images of handwritten and printed equations, handwritten and printed text, tables, and diagrams to get digital formats like LaTeX, Asciimath, and MathML.
